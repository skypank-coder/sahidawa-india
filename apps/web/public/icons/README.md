# PWA Icons

Place the following icon files in this directory:

- `icon-192.png` — 192×192 px PNG (used for Android home screen, PWA install)
- `icon-512.png` — 512×512 px PNG (used for splash screen, app store listing)

Both should use the SahiDawa logo on a white or green (#16a34a) background.
The icons should be maskable (safe area: centered logo within 80% of canvas).

You can generate these from the main logo SVG using:
  https://maskable.app/editor
